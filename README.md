# imcache

In-Memory cache for node.js

### Install

To install imcache in an existing project as a dependency:

Install with npm:
```sh
npm install imcache
```
Install with yarn:
```sh
yarn add imcache
```
Install with pnpm:
```sh
pnpm install imcache
```

### Example

```js
// ESM
import imcache from 'imcache'
const cache = new imcache()

// CommonJs
const imcache = require('imcache')()
```
